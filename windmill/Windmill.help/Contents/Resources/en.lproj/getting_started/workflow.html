Workflow

You can add additional branches for Windmill to monitor.

Make sure your project repository is already at the branch you want to monitor.

  1. Open the File menu
  2. Press and hold the Option key
  3. Select Open at Branch...
  4. Select the Xcode project/workspace
Tip: Use ⎇⌘O to quickly open a project at a branch.

For any additional branch to master, Windmill will provide you with error and test failure highlighting.

When you are done working on a feature branch, Close the Window (⌘W) to have Windmill stop monitoring the branch.

Tip: Under the 'Window' menu, select 'Move Tab to New Window' to detach a branch you monitor and make it full screen.

Monitor a feature branch and master.

It is recommended that Windmill always monitors the master branch so that you always know you can install your app on a device and release to the App Store.

Tip: Use ⇧⌘\ to toggle between a single branch focus and the overview.
